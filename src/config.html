Cybozu Live Notifier 初期設定

Cybozu Live Notifierを利用するためには、初回に
  1. サイボウズLive Developer Centerへのアカウント登録
  2. アプリケーション登録によるConsumer Key、Consumer Secretの取得
  3. Access token、Access token secretの取得
の3つの作業が必要です。

サイボウズLive Developer Centerへのアカウント登録

サイボウズLive Developer Centerよりデベロッパーとしてのアカウント登録を行って下さい。

アプリケーション登録によるConsumer Key、Consumer Secretの取得

アプリケーションの登録 - サイボウズLive Developer Center より新しくアプリケーションを登録してください。
アプリケーション名には任意の名前を、アプリケーションの種類には「クライアント」を、アクセスレベルには「レベルA」を選択し、アプリケーションを登録します。

Access token、Access token secretの取得

Myアプリケーション - サイボウズLive Developer Center から登録したアプリケーションを選択し、表示されているConsumer Key、Consumer Secretを以下のフォームに入力します。 普段、サイボウズLiveで使用しているメールアドレス、パスワードとともに以下のフォームに入力します(パスワードはCybozu Live Notifierでは保存されません)。 「Access Tokenを取得」をクリックしAccess token、Access token secretが取得できれば終了です。
